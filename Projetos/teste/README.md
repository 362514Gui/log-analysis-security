🔐 Security Log Analyzer

Este projeto analisa logs de autenticação para identificar comportamentos suspeitos, como múltiplas tentativas de login em um curto período de tempo.

🚀 Funcionalidades

- Detecção de tentativas de login suspeitas
- Análise de padrões de comportamento
- Alertas de possível ataque de força bruta

🛠️ Tecnologias utilizadas

- Python
